## 功能大項
1. 登入前（會有 立即加入 按鈕）
2. 註冊（標題：註冊；會有以下欄位：姓名、生日、性別、身分證字號、使用者帳號、密碼、地址、手機號碼、信箱）、
3. 登入畫面（標題：登入；會有以下欄位：使用者帳號、密碼（要有遮罩功能）；底下有 登入、一鍵帶入 兩個按鈕）
4. 客戶端-首頁（頂部bar的右邊，會有一個圓形的大頭照按鈕、「歡迎，XXX（使用者姓名）」、 登出 按鈕；點按圓形大頭照後會進入「會員中心」
5. 客戶端-會員中心（個人資料修改頁面：左邊是一個圓形大頭照（支援大頭照上傳 (Avatar Upload)）、
   底下有客戶編號（cif）（這兩個欄位看起來像一組，較窄）、右邊是：姓名、生日、性別、使用者帳號、密碼、手機號碼、信箱、地址
6. 系統管理員端 -
   Level 0：職員：可以對「客戶表」進行：增、停用（軟刪除）、查、改
   Level 1：超級管理員：可以對「客戶表」進行：增、停用（軟刪除）、查、改
